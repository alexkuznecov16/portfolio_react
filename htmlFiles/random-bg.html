Random background color

In this project has been used an HTML, CSS and JS technologies. In JavaScript I created a "symbols" var, where you can see al symbols for HEX-color and "color" var with value #. My algorithm is next. Loop with name "FOR" execute a code 6 times. When he executed variable "COLOR" adds after # random symbol from "SYMBOLS" variable. On the end "body" has been changed a background color, where value was a "COLOR" var value (# + 6 random symbols) + the new random generated color was add in html text, that user can to sees a HEX code of color.
